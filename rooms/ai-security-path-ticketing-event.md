## What this room teaches

A short orientation room TryHackMe published to mark the launch of their **AI Security learning path**. It explains how the launch event works, the structure of the path (25 rooms across 5 modules), and how to engage with the live community elements during the event.

## Key takeaways

- The AI Security path is structured into 5 modules: AI/ML threats, AI models and data, prompt engineering, LLM security, and AI threat modeling — plus deeper rooms on AI forensics, supply chain security, and RAG security.
- This room is essentially an event RSVP and orientation, not technical content. The technical learning starts in the actual path rooms.
- The "AI Security as a separate learning path" framing is itself worth noting: TryHackMe is treating AI security as its own track, not just an add-on to existing red/blue paths.

## What was new to me

- AI security is a genuinely new discipline — the curriculum was being written in real time during the launch window. That's both a risk (less-mature material) and an opportunity (early entrants build a credible portfolio while the field is still defining itself).
- Per the path framing, AI security isn't a replacement career — it's a **layer on top of** an existing specialization. A SOC analyst who learns AI security becomes a SOC analyst who can investigate AI incidents. That's how I'm planning to position it: foundations first, AI security as a year-2 specialization.

## How a SOC analyst would use this

- Knowing the AI Security path exists matters even if I'm not actively in it right now. By the time I'm interviewing for SOC roles in mid-2026, hiring managers will be asking whether candidates have any AI exposure. Having "AI Security path completed" on my profile (even partially) signals awareness without overcommitting.
- A SOC team in 2026+ is going to have to triage alerts that involve AI assistants, AI-augmented phishing, and prompt-injected agentic systems. Even the orientation room reinforces that this is now part of the SOC's scope, not a separate specialty.
